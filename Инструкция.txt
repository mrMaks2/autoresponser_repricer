Данное приложение осуществляет функции автоответа на необработанные отзывы кабинета 3 с оценками 4 и 5

Для запуска проведите следующие операции:

 1. Установите Python в Ваш ОС. Если устанволен, переходите к следующему пункту
 2. Если Python установлен, но нет в файле виртуального окружения (venv):
    2.1 Запустите командную строку CMD
    2.2 Перейдите на директорию приложения (C:\auto_responser) при помощи команды cd
    2.3 Установите вритуальное оркужение командой python -m venv venv
 3. Активируйте вритуальное оркужение при помощи команды venv\Scripts\activate
 4. Установите зависимости при помощи команды pip install -r requirements.txt
 5. Запустите приложение при помощи команды python main.py
